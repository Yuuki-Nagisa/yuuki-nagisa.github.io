# Portfolio Website

This project is a modern, component-based portfolio website built using React.js, Tailwind CSS, and Framer Motion. The website is designed to showcase skills, projects, and experience with a strong focus on maintainability, performance, and user experience.

## Tech Stack
- **React.js:** For building reusable, component-driven UI.
- **Tailwind CSS:** Utility-first CSS framework enabling rapid styling with responsive design.
- **Framer Motion:** For smooth, performant animations and interactive UI transitions.

## Features
- **Component-Based Architecture:** Modular React components improve maintainability, reusability, and accelerate development.
- **Responsive Design:** Desktop-first approach with distinct layouts and interaction patterns tailored for both desktop and mobile devices.
- **Animations:** Utilized Framer Motion to create engaging animations that enhance UX while balancing frontend performance.
- **Performance Optimization:** Applied Chrome Lighthouse audits to optimize load speed, accessibility, and overall site performance.
- **SEO Best Practices:** Implemented meta tags, structured data, and configured Google Search indexing to improve discoverability.
- **Deployment:** Hosted on GitHub Pages with a custom domain configured via Namecheap DNS, ensuring reliable and accessible hosting.

## Getting Started

### Prerequisites
- Node.js (v14+ recommended)
- npm or yarn package manager

### Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/portfolio-website.git
    ```
2. Navigate to the project directory:
    ```bash
    cd portfolio-website
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
    or
    ```bash
    yarn install
    ```

### Running Locally
Start the development server with:
```bash
npm start
